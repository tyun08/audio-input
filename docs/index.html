Features

Everything you need.
Nothing you don’t.

Built to stay out of your way. Invoke it, speak, and get back to work.

Works Everywhere

Injects text directly into any focused input via macOS Accessibility API. Slack, Notion, Terminal, browsers — anywhere.

AI-Powered Polish

An optional LLM pass removes filler words, fixes punctuation, and cleans up sentence flow. Screenshot context makes it smarter.

Privacy First (BYOK)

Bring Your Own API Key. Audio goes directly to Groq — no server of ours ever sees it. Unlike closed-source alternatives, you can read every line of code.

50+ Languages

Powered by Whisper large-v3-turbo. Chinese, English, Japanese, Korean, Spanish and more — auto-detected.

Featherweight

~20 MB RAM. Built with Tauri + Rust. No Electron, no background bloat. It just sits in your menu bar.

Open Source

MIT licensed. Audit the code, fork it, extend it. No subscription, no upsell. A better alternative to SuperWhisper.

Setup

Up and running in 3 steps

No account needed. Just a free API key.

1

Get a free Groq API key

Sign up at console.groq.com. The free tier is generous enough for daily use.

2

Set your key in the app

Launch Audio Input. Right-click the menu bar icon and choose Configure API Key, then paste your Groq key.

3

Press ⌘⇧Space and speak

Hold the hotkey, dictate, release. Your words appear at the cursor in any app — polished and ready.
