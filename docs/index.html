How it works

1

Get a free Groq API key

Sign up at console.groq.com — free tier is more than enough.

2

Install and paste your key

Homebrew or DMG. Right-click the menu bar icon → Configure API Key.

3

Press ⌘⇧Space and speak

Release the hotkey. Text appears at your cursor in under a second.
